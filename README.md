Donut Haven - Online Food Ordering

Live Website URL: https://donut-haven.vercel.app

About
A modern, responsive e-commerce platform for ordering delicious donuts and baked goods online. Donut Haven provides a seamless ordering experience with real-time cart management, secure checkout, and order tracking.

Features
- Browse donuts with photos and descriptions
- Add items to cart and manage quantities
- Secure checkout process
- Order tracking

Responsive Design
Donut Haven is fully responsive across all devices:
 1. Desktop: Optimized for large screens with multi-column layouts
 2. Tablet: Adaptive navigation and product grids
 3. Mobile: Touch-friendly interfaces and mobile-optimized checkout
